# wiggly_loaders example

Example app for the `wiggly_loaders` package.

## Run

```bash
flutter pub get
flutter run
```

## Included demo states

- `WigglyLoader` determinate with slider-controlled progress
- `WigglyLoader.indeterminate()`
- `WigglyLinearLoader` determinate
- `WigglyLinearLoader.indeterminate()`
- `WigglyRefreshIndicator` pull-to-refresh demo

## Mobile targets

Native folders are included for:

- Android
- iOS
